HTML

HTML是超文本标记语言,我们可以这样认为,HTML就
是普通的文本文件,只不过在文本中的内容如果被一些
特殊的标签进行包裹就有了特殊的含义,这些被那些标
签标记文本,就成了超文本
我们平时在浏览器中浏览的网页就是就是使用HTML
语言进行编写的

网页的组成

网页的内容组成

  1. HTML 用于展示需要显示的信息
  2. CSS 用来美化我们写的页面
  3. JavaScript 是页面具有一定的交互效果

根据内容的划分,可以将网页划分为静态页面和动态页
面

  1. 静态页面就是编写以后在浏览器中不再改变的网
    页,如HTML
  2. 动态页面就是可以根据不同的情况显示不同的内
    容,如jsp,php,动态页面也是在HTML的基础上添加
    的一些内容

HTML的结构

  • HTML不需要编译,直接用浏览器阅读即可
  • 扩展名是.html或者.htm
  • 由标签组成
    • 标签不区分大小写
    • 单标签:以/结尾如: ,/也可以不写
    • 双标签:有开始标签有结束标签

标签

h系列标签

font标签

设置文本的大小颜色等信息,不会独占一行
size属性:1­ 7,默认是3
color属性:可以是#xxxxxx表示3原色,也可以是
red.blue,green等
如果红绿蓝2位取值相同,可省略成1位,例如#FF1133
简化为#F13

p标签

段落标签,会单独占据一行

hr标签

分割线标签,会在页面上显示一条分割线
size属性:水平线的高度

b标签

粗体标签,不会独占一行

span标签

显示文本的,不会单独占据一行
